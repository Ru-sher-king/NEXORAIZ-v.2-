# NEXORAIZ

Aplicación educativa e interactiva que fomenta el turismo nicaragüense, inspirada en Duolingo para el aprendizaje de los idiomas indígenas:

- Mayagna
- Miskitu

## Funciones

- Sistema de niveles
- Vidas
- Puntuación
- Desbloqueo progresivo
- Interfaz interactiva

## Tecnologías

- HTML5
- CSS3
- JavaScript
